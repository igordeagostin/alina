# Preferências e convenções (exemplo)

Copie este arquivo para o caminho configurado em `Storage:PreferencesFile`
(padrão: `%APPDATA%/Alina/preferences.md`) e ajuste ao seu gosto. Todo o
conteúdo é injetado no system prompt da Alina.

## Sobre mim

- Desenvolvedor .NET, trabalho principalmente com C#.
- Respostas objetivas e em português do Brasil.

## Convenções de código

- Clean Architecture, nomes em português no domínio.
- Testes com xUnit.
- Sempre rodar os testes antes de considerar uma tarefa concluída.

## Segurança
